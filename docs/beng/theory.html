2.3 Fiduciary Markers

2.3.1 Introduction******

The other broad area involved in this project is fiduciary marker identification and analysis. Fiduciary markers are distinguishable parts of a physical body that can be used to help determine positions of real features from images.  

For fiduciary markers to be useful, they need to be easily identifiable, and something about how the markers represent parts of the physical body (such as real-life distance between two markers), needs to be known.

2.3.2 Marker Identification By Colour

For a computer to recognise a marker in an image, something about the visual properties of the marker needs to be identifiable, by the computer, from those of the rest of the image. One of the most common approaches is to make fiduciary markers of a distinct colour, not naturally present on the object of interest.

Obviously, this involves using colour images, rather than greyscale images. A bitmap (with a ‘.bmp’ file extension) image file stores a colour image M by N pixels in size as a M by N by 3 matrix[3]. Each of the three positions of the third dimension stores the red, green and blue component (respectively) of the colour image.

For example, if the colour image is:

[Fig. 2.3.2]

Colour image taken using a simple webcam

[Fig. 2.3.3]

Red Green

Blue**

Red, green and blue components of the colour webcam image

[Fig. 2.3.4]

Different combinations of the three stored colour intensity planes produce other intensity planes whose intensity is greatest only at specific colours. For example, a red colour has a high intensity value on the red intensity plane, but a white area also has a high intensity value on the red intensity plane (as white light is a combination of all colours of visible light). By subtracting the average (greyscale) intensity of all three planes from the red intensity plane, a resulting intensity plane is created where the greatest intensity values occur only at parts of the image that are exclusively red; thus it takes information from all three colour planes to determine areas of a specific colour:

Exclusive* red intensity planes for both test images: Note the increased contrast compared with the non-exclusive red intensity images, which would make identifying markers of the specific colour easier.*

[Fig. 2.3.5]

This kind of processing is useful in determining marker positions based on colour. A first stage in identifying a marker could be to find the areas of greatest intensity in the exclusive intensity plane of the colour of the marker, although more processing needs to take place to make sure the marker is correctly identified.

A fiduciary marker need not be of a red, green or blue colour, as any colour can be identified using the right combination of red, green and blue intensity planes. Even black can be used- by creating a marker intensity plane that is the inverse of the greyscale intensity plane (itself the average of the three colour intensity planes) – although using a marker on the greyscale is not advisable in real-life situations, as both very bright and very dark areas of images can be the result of lighting, rather than object colour.

A simple example demonstrating the problems with using black or white as marker colours: It would be difficult for a computer to identify the headphones from the surrounding area as the shadows formed nearby are of a similar colour. This would not be the case if the headphones were, for instance, yellow.

[Fig. 2.3.6]
